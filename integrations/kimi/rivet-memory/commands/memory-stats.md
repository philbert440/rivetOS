# /memory-stats

Quick health check and overview of the RivetOS memory store.

**Usage:**
```
/memory-stats
```

Shows conversation counts, message volume, coverage by agent (`rivet-claude`, `rivet-hermes`, `rivet-grok`, `rivet-kimi`, etc.), compaction status, and embedding queue health.

Useful before large recall tasks or when diagnosing thin memory results.
